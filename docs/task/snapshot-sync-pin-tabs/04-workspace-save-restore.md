# 子任务 04：工作空间保存与恢复 MVP

## 目标

让用户一键保存当前浏览器窗口为工作空间，并能从工作空间页面、地址栏搜索等入口找回和恢复。

## 范围

- 定义工作空间数据模型。
- 保存当前窗口打开的页面。
- 支持保存弹窗和保存完成反馈。
- 提供工作空间列表、详情、搜索和筛选。
- 支持恢复工作空间。
- 支持地址栏搜索工作空间并恢复。

## 不包含

- 工作空间转书签。
- 工作空间 AI 深度分析。
- Tab 自动分组。
- 工作空间置顶。

## 数据模型

工作空间包含：

- 工作空间名称。
- 描述。
- 分类。
- 标签。
- 保存时间。
- 页面列表。
- 每个页面的标题、URL、域名、图标。
- AI 分析结果预留字段。
- 是否已恢复。
- 是否已转为书签。

## 保存入口

支持：

- 快捷键：保存当前窗口打开的所有页面。
- 右键菜单：保存当前窗口为工作空间。
- 扩展菜单：保存当前窗口。
- 工作空间页面：`保存当前窗口`。
- 侧边栏快捷入口：`保存当前工作空间`。

## 保存范围

默认：

- 保存当前窗口打开的所有页面。

可选：

- 保存所有窗口。
- 保存当前标签页左侧。
- 保存当前标签页右侧。
- 手动选择部分页面。
- 排除浏览器固定标签页。
- 保存后关闭原标签页。

V1 建议：

- 默认保存当前窗口。
- 默认不关闭原标签页。
- 默认保留浏览器固定标签页。

## 保存弹窗

字段：

- 工作空间名称。
- 描述。
- 分类。
- 标签。
- 页面列表。
- 重复 URL 提示。
- AI 分析开关。
- 保存按钮。

名称生成：

- 用户可手动输入。
- 可点击 `AI 生成名称`。
- 如果用户不输入，使用时间和主要域名生成临时名称。

保存完成反馈：

- 成功：`工作空间已保存`，可点击 `查看工作空间` 或 `继续浏览`。
- 失败：展示失败原因，保留已填写内容，并提供重试。

## 搜索与恢复

地址栏搜索范围：

- 工作空间名称。
- 工作空间描述。
- 分类。
- 标签。
- 页面标题。
- 页面域名。

地址栏结果展示：

- 工作空间名称。
- 页面数量。
- 最近保存时间。
- 主要域名或分类。

工作空间页面支持：

- 搜索。
- 按分类筛选。
- 按标签筛选。
- 按保存时间排序。
- 按最近恢复排序。
- 查看工作空间详情。

恢复方式：

- `在新窗口恢复`
- `在当前窗口恢复`
- `只恢复选中页面`

恢复前确认：

- 页面数量较多时提示用户确认。
- 当前窗口已有重复 URL 时提示是否跳过重复项。

## 状态

- `已保存`
- `已恢复`
- `包含重复页面`

## 验收标准

- 用户能从至少一个入口保存当前窗口所有页面。
- 保存弹窗能展示页面列表并允许编辑基础信息。
- 保存成功后能在工作空间列表看到记录。
- 用户能搜索、筛选和打开工作空间详情。
- 用户能在新窗口恢复工作空间。
- 页面数量较多时恢复前有确认。
- 地址栏能搜索到工作空间并触发恢复。
- 恢复完成后更新最近恢复时间。

## 依赖

- 浏览器 Tabs API。
- 扩展快捷键和右键菜单能力。
- 持久化存储。
- 地址栏搜索入口。
