# 人工测试用例：03 分类和书签置顶

## 前置条件

- 已安装开发版 HamHome 扩展。
- 至少存在 2 个分类和 3 个书签。
- 能打开书签管理页、分类管理页和内容页侧边栏。

## 用例 1：书签管理页置顶书签

步骤：

1. 进入书签管理页。
2. 找到任意书签，打开更多菜单。
3. 点击 `置顶`。
4. 再次打开该书签更多菜单。

预期：

- 操作后提示已置顶。
- 菜单文案变为 `取消置顶`。
- 打开内容页侧边栏后，顶部出现 `置顶` 区，并展示该书签标题和域名。

## 用例 2：书签取消置顶

步骤：

1. 在书签管理页找到已置顶书签。
2. 打开更多菜单，点击 `取消置顶`。
3. 打开内容页侧边栏。

预期：

- 操作后提示已取消置顶。
- 如果没有其他置顶内容，侧边栏不显示 `置顶` 区。
- 如果仍有其他置顶内容，已取消的书签不再显示。

## 用例 3：分类管理页置顶分类

步骤：

1. 进入分类管理页。
2. 找到任意分类，打开更多菜单。
3. 点击 `置顶`。
4. 打开内容页侧边栏。

预期：

- 侧边栏 `置顶` 区展示该分类名称、图标或文件夹图标，以及该分类下书签数量。
- 分类管理页该分类菜单文案变为 `取消置顶`。

## 用例 4：点击置顶分类筛选书签

步骤：

1. 侧边栏中点击一个置顶分类。

预期：

- 侧边栏书签列表切换为该分类下的书签。
- 搜索关键词被清空。
- 不打开新网页。

## 用例 5：点击置顶书签打开网页

步骤：

1. 侧边栏中点击一个置顶书签。

预期：

- 浏览器打开该书签 URL。
- 侧边栏不会错误切换分类筛选。

## 用例 6：最近置顶排在前面

步骤：

1. 依次置顶书签 A、书签 B、分类 C。
2. 打开内容页侧边栏。

预期：

- 默认排序为分类 C、书签 B、书签 A。

## 用例 7：手动排序置顶内容

步骤：

1. 打开内容页侧边栏。
2. 鼠标悬停置顶内容。
3. 点击某个置顶项的上移或下移按钮。
4. 关闭并重新打开侧边栏。

预期：

- 置顶项顺序立即变化。
- 重新打开侧边栏后顺序保持不变。

## 用例 8：置顶内容较多时折叠

步骤：

1. 置顶超过 5 个分类或书签。
2. 打开内容页侧边栏。
3. 点击置顶区右侧展开/折叠按钮。

预期：

- 初始只展示前 5 个置顶项。
- 点击展开后显示全部置顶项。
- 点击折叠后恢复只显示前 5 个。
