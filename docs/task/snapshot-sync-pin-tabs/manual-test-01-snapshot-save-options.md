# 人工测试用例：01 快照保存可选

## 前置条件

- 已安装开发版 HamHome 扩展。
- 至少打开一个普通网页。
- 能进入 HamHome 设置页和书签管理页。

## 用例 1：设置页默认保存快照开启

步骤：

1. 打开设置页，进入 `通用设置`。
2. 开启 `默认保存快照`。
3. 将 `默认快照类型` 设为 `自动`。
4. 打开任意网页，打开保存面板。

预期：

- 保存面板中的 `保存快照` 开关默认开启。
- 快照类型显示为 `自动`。

## 用例 2：设置页默认不保存快照

步骤：

1. 打开设置页，进入 `通用设置`。
2. 将 `默认快照类型` 设为 `默认不保存`。
3. 打开任意网页，打开保存面板。

预期：

- 保存面板中的 `保存快照` 开关默认关闭。
- 保存书签后，书签可创建成功。
- 该书签在管理页不显示已有快照状态。

## 用例 3：保存面板临时关闭快照

步骤：

1. 设置页保持 `默认保存快照` 开启。
2. 打开保存面板。
3. 关闭保存面板中的 `保存快照` 开关。
4. 保存书签。
5. 再次打开设置页。

预期：

- 本次书签保存成功，且未保存快照。
- 设置页中的 `默认保存快照` 仍保持开启，没有被本次临时选择改写。

## 用例 4：保存面板保存快照

步骤：

1. 设置页开启 `默认保存快照`。
2. 打开一个可读网页。
3. 打开保存面板，确认 `保存快照` 开启。
4. 保存书签。
5. 进入书签管理页，打开该书签更多菜单。

预期：

- 保存过程中能看到书签/快照保存状态变化。
- 书签管理页中该书签更多菜单包含 `查看快照`、`更新快照`、`删除快照`。
- 点击 `查看快照` 能打开快照查看器。

## 用例 5：书签管理页保存或更新快照

步骤：

1. 找到一个没有快照的书签。
2. 打开更多菜单，点击 `保存快照`。
3. 等待成功提示。
4. 再次打开更多菜单，点击 `更新快照`。

预期：

- 第一次操作后书签标记为有快照。
- 第二次操作后仍保留快照，且没有创建重复书签。
- 快照操作失败时展示失败提示，不影响书签本身。

## 用例 6：书签管理页删除快照

步骤：

1. 找到一个已有快照的书签。
2. 打开更多菜单，点击 `删除快照`。
3. 确认删除。

预期：

- 快照删除成功后展示成功提示。
- 该书签更多菜单不再展示 `查看快照` 和 `删除快照`。
- 书签本身仍保留在列表中。
