# 子任务 06：工作空间智能分析

## 目标

在保存或查看工作空间时分析 Tab，帮助用户理解工作现场、生成名称和标签、筛选可转书签页面，并处理完全重复 URL。

## 范围

- 工作空间页面分类。
- 工作空间名称和标签推荐。
- 完全相同 URL 去重。
- 可转书签页面建议。
- 分析结果展示。

## 不包含

- 工作空间保存与恢复基础流程。
- 转书签写入逻辑。
- 新标签页自动分组。

## 分析用途

分析结果用于：

- 给工作空间命名。
- 给工作空间打标签。
- 给页面分组。
- 转书签时筛选页面。
- 清理重复页面。

## 分类规则

工作空间内页面分类优先级：

1. 优先使用浏览器 Tab 分组名称。
2. 如果没有 Tab 分组，则使用用户已有 Tab 分组规则。
3. 如果仍未匹配，则可使用 AI 进行分类。
4. 如果用户关闭 AI 分类，则归入 `未分类`。

推荐分类示例：

- `参考文档`
- `技术文档`
- `搜索结果`
- `管理平台`
- `代码仓库`
- `设计素材`
- `沟通协作`
- `待阅读`
- `未分类`

## 完全相同 URL 去重

工作空间保存或分析时自动识别完全相同的 URL。

交互：

- 保存弹窗提示：`发现 3 个重复页面`
- 默认保留一个。
- 用户可展开查看重复项。
- 用户可选择保留哪个页面。
- 用户可选择仍然全部保存。

V1 建议：

- 对完全相同 URL 自动去重。
- 去重前展示提示。
- 用户可以撤销或修改。

## 智能分析展示

工作空间详情页展示：

- 页面总数。
- 去重后页面数。
- 页面分类分布。
- 推荐标签。
- 推荐名称。
- 可转书签页面建议。
- 被排除页面建议。

## 数据与状态

建议记录：

- 页面分类。
- 页面用途说明。
- 是否重复 URL。
- 重复组 ID。
- 推荐工作空间名称。
- 推荐标签。
- 转书签建议。
- 排除原因。
- 分析时间。

## 验收标准

- 工作空间保存或详情页能触发分析。
- 分类优先级符合规则：浏览器 Tab 分组、用户规则、AI、未分类。
- 完全相同 URL 能被识别并提示。
- 默认去重前用户能看到重复提示。
- 用户能选择保留重复页或全部保存。
- 详情页能展示分类分布、推荐名称、推荐标签和转书签建议。
- 关闭 AI 分类时，未匹配页面进入 `未分类`。

## 依赖
