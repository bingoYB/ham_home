核心体验

  • 自动生成标题摘要，快速判断内容价值

  • 支持自带 API Key，也能配置隐私域名跳过分析

收藏管理，从随手调用到集中整理都连贯

  • 任意页面都能通过侧边栏快速调出书签
  • 集中管理页支持卡片 / 列表切换、筛选与批量操作
  • 对话式搜索能继续追问、缩小结果范围

把当前浏览现场，直接收成可恢复的工作空间

  • 保存当前窗口，保留页面组合与原生 Tab Groups
  • 按规则或 AI 自动完成 Tab 分组，减少手工整理
  • 适合研究资料、项目上下文和周期性浏览任务
