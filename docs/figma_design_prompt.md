# HamHome Figma 设计提示词

## 项目概述
HamHome 是一个 AI 驱动的智能书签管理工具，包含浏览器插件和 Web 管理端。设计风格应简洁现代，支持明暗主题切换。

---

## 浏览器插件端设计

### 1. Popup 面板 - 保存书签界面
**尺寸**: 400px × 500px

**功能模块**:
- 顶部导航栏：Logo + 视图切换按钮（收藏/列表）
- AI 分析状态提示条（加载中/成功/失败/未配置）
- 表单区域：
  - 标题输入框
  - 摘要文本域（多行）
  - 分类下拉选择器（支持新建）
  - 标签输入框（支持多标签，回车添加）
- 底部操作按钮：保存/更新、重新分析（AI）

**状态**:
- 已收藏提示（显示"此页面已收藏，可更新信息"）
- AI 分析中加载态
- 表单验证错误态

---

### 2. Popup 面板 - 书签列表视图
**尺寸**: 400px × 500px

**功能模块**:
- 顶部搜索框
- 书签卡片列表：
  - Favicon
  - 标题
  - 摘要（截断显示）
  - 标签（Badge 形式）
  - 创建时间
  - 悬停操作按钮（编辑、删除）

**状态**:
- 空状态（无书签提示）
- 加载状态
- 搜索结果高亮

---

### 3. Options 设置页面
**尺寸**: 全屏页面（建议最大宽度 1200px）

**标签页结构**:
- **AI 配置标签页**:
  - 启用 AI 分析开关
  - AI 服务商选择（OpenAI/Anthropic/Ollama/自定义）
  - API Key 输入框（密码类型）
  - Base URL 输入框（条件显示）
  - 模型名称输入框
  - 高级参数折叠面板（Temperature、Max Tokens）
  - 保存/测试连接按钮

- **通用设置标签页**:
  - 自动保存快照开关
  - 主题选择（跟随系统/明亮/暗黑）
  - 语言选择（中文/English）

- **存储管理标签页**:
  - 快照存储信息（数量、占用空间）
  - 清除所有快照按钮
  - 数据导出按钮（JSON/HTML）

---

## Web 端设计

### 4. 登录/注册页面
**布局**: 居中卡片式

**功能模块**:
- 标题 Logo
- 邮箱/密码输入框
- 登录按钮
- 第三方登录按钮（Google、GitHub）
- 注册/忘记密码链接

---

### 5. Dashboard 主页面 - 书签列表
**布局**: 侧边栏 + 主内容区

**侧边栏**:
- Logo
- 导航菜单：全部书签、回收站、导入、设置
- 分类列表（文件夹图标）
- 标签云（可选折叠）

**主内容区**:
- 顶部搜索栏（支持快捷键 ⌘K）
- 视图切换按钮（卡片视图/列表视图）
- 批量操作栏（选中时显示）
- 书签展示区：
  - **卡片视图**: 网格布局（响应式 1-3 列），卡片包含封面图、标题、摘要、标签、操作按钮
  - **列表视图**: 表格布局，紧凑显示

**状态**:
- 空状态
- 加载骨架屏
- 错误状态

---

### 6. 书签详情/编辑页面
**布局**: 侧边栏 + 详情面板

**功能模块**:
- 书签元信息展示（标题、URL、摘要、分类、标签）
- 编辑表单（与 Popup 保存面板类似）
- 操作按钮：编辑、删除、打开原链接、查看快照
- 正文预览区域（Markdown 渲染）

---

### 7. 分类管理页面
**功能模块**:
- 分类树形列表
- 新建分类按钮
- 分类操作（编辑、删除、移动）

---

### 8. 标签筛选页面
**功能模块**:
- 标签云展示
- 点击标签筛选书签列表
- 标签管理（合并、删除）

---

### 9. 回收站页面
**功能模块**:
- 已删除书签列表
- 恢复按钮
- 永久删除按钮

---

### 10. 搜索页面
**功能模块**:
- 搜索输入框（支持实时搜索）
- 搜索结果列表
- 搜索类型标识（关键词/语义/混合）

---

### 11. 书签导入页面
**功能模块**:
- 文件上传区域（拖拽上传）
- 导入预览（显示解析结果：数量、文件夹结构）

  - 保留文件夹/转为标签
  - 是否启用 AI 处理

- 导入结果统计（成功/跳过/失败）

### 12. 设置页面
**标签页结构**:

## 设计规范

### 颜色系统
- 主色调：品牌色（建议使用温暖的橙色或蓝色）
- 支持明暗主题切换
- 语义化颜色：成功（绿）、警告（黄）、错误（红）

### 字体
- 标题：粗体，较大字号
- 正文：常规，适中字号
- 辅助文字：较小字号，降低对比度

### 间距
- 统一使用 4px/8px 基础间距系统
- 卡片内边距：16px
- 页面边距：24px

### 组件库
- 按钮：主要/次要/文本按钮
- 输入框：标准/密码/搜索
- 卡片：带阴影，圆角
- 标签/Badge：小圆角，不同颜色变体
- 对话框/Modal：居中，遮罩层

**MVP 阶段优先设计**:
1. Popup 保存书签界面
2. Popup 书签列表视图
3. Options 设置页面（AI 配置）
4. Web 端登录页面
5. Web 端 Dashboard 主页面（书签列表）

**后续版本设计**:
- 其他 Web 端页面
- 高级功能界面
